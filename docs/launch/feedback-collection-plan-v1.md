# Curavon Feedback Collection Plan v1

For early real-user testing and demo follow-ups.

---

## Questions for Every Tester

1. Did you understand what Curavon does?
2. Did the next action on Today feel useful?
3. Did the safety message feel clear — not scary?
4. Did Ask Curavon feel like it understood your concern?
5. Did Guides feel useful or confusing?
6. Would you use Doctor Summary before a clinic visit?
7. Did anything feel like medical advice?
8. Did anything feel unsafe?
9. Was the app too much, too little, or just enough?
10. What would make you come back tomorrow?

---

## How to Collect

- **During session:** Observer notes confusion points; do not lead the user.
- **After session:** 5-minute verbal debrief using questions above.
- **Optional:** Screen recording with consent for UX review only.

---

## Red Flags to Escalate Immediately

- User believes Curavon diagnosed them
- User followed a self-care step during an urgent safety scenario
- User cannot find export/delete controls
- User thinks data is backed up to cloud when it is not
